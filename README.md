# Flow CLI

## Commands

* **Default run** (no arguments).
  - Pomodoro Mode.
  - Focus: 25 minutes.
  - Break: 5 minutes.

* **Custom run** (with arguments).
  - Required: `flow <name> <minutes>`.
  - Optional: `--break <minutes>` or `-b <minutes>`.
    - If you skip it, there wonâ€™t be a break.
